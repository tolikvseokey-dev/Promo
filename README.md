# Analytics Telegram Bot

Telegram-бот для управленческой аналитики сети.

## Возможности
- MTD аналитика сети (ТО Факт / ТО План на дату)
- Выполнение плана по региональным менеджерам
- LFL сравнение (текущий год vs прошлый)
- ТОП / АНТИ-ТОП-3 лавок по LFL
- Неделя к неделе (LFL) для текущего и прошлого года
- Единый отчёт одним сообщением в Telegram

## Источники данных
Excel-файлы загружаются в бот через Telegram и **не хранятся в репозитории**.

Обязательные файлы:
- ТО 25 / ТО 26
- чеки 25 / чеки 26
- длина 26
- планы
- ростер

(файлы `ср чек` можно не загружать — средний чек считается как ТО / Чеки)
